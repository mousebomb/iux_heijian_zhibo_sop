# 黑键直播多维表格脚本

功能： 对一条直播排期 一键自动分配直播任务

 - 脚本入口： https://mousebomb.github.io/iux_heijian_zhibo_sop/
 - 技术支持： https://digitalforces.iuxlabs.com/flight.html

# Getting Started
- Hit run
- Edit [runUIBuilder.ts](#src/runUIBuilder.ts) and watch it live update!
